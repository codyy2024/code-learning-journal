# Day 71 Part 2: Basic CSS - Inline-Block

Inline element: you can't adjust the height and width + flow with surrounding text and don't start a new line

Block element: you can adjust the height and width (default == full width) + always start a new line

Inline-block: you can adjust the height and width + flow with surrounding text and don't start a new line
