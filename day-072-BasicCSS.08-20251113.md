# Day 72: Basic CSS - Margins and Padding

Margin properties
- apply spacing **outside** the element so it separates itself from the other elements

Padding properties
- apply spacing **intside** the element so it separates the content inside from the border of the element

Things they share in common:
- you can use `margin-top`, `margin-bottom`, `margin-left`, `margin-right`, or `padding-top`, `padding-bottom`, `padding-left`, `padding-right`,
- you can also specify one, two, three, or four values at once (shorthand)
  - one, example `margin: 10px`: applies to all four
  - two, example `margin: 10px 20px;`: applies `10px` to top and bottom, and `20px` to left and right
  - three, example `margin: 10px 20px 30px;`: applies `10px` to top, `20px` to right, `30px` to bottom
  - four: example `margin: 10px 20px 30px 40px;`: applies `10px` to top, `20px` to right, `30px` to bottom, and `40px` to left (basically starting from top and goes clockwise)
